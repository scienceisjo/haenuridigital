🎯 게임 소개
📖
이 게임은 해누리중학교 디벗(디지털 벗)을 올바르게 사용하는 방법을 배우는 교육용 체험 게임입니다.
🗺️
학교 맵을 자유롭게 돌아다니며 14가지 디벗 관련 상황을 만나고, 올바른 선택을 해보세요!
⭐
정답을 맞추면 +15점을 얻고, 틀리면 HP가 깎입니다. HP가 0이 되면 게임 오버!
🕹️ 조작 방법
⬆⬇⬅➡
방향키 / D-패드로 맵의 미션 노드를 이동합니다.
✅
Enter 버튼을 누르면 현재 위치의 미션에 도전합니다. (노드가 황금색으로 빛날 때!)
👆
모바일에서는 노드를 직접 터치해서 이동하거나, 화면 오른쪽 아래 D-패드로 이동하세요.
1️⃣2️⃣
선택지 팝업에서 키보드 숫자 1, 2 또는 위아래 방향키로 답을 고를 수 있어요.
